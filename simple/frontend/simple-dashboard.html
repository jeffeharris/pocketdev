🚀 Claude Terminal

Open Claude Terminal

Direct access to Claude through a web terminal. No complex monitoring or proxies.

📊 Monitor Claude (Optional)

If you want notifications when Claude needs attention, run this in a terminal:

./simple-monitor.sh

With webhook notifications (for Slack, Discord, etc):

./simple-monitor.sh "project-name" "https://your-webhook-url"

🎯 Quick Tips

  • Multiple Projects: Run Claude on different ports:
    docker run -d -p 7682:7681 -v /path/to/project:/workspace claude-ttyd
  • Check Status:
    ./claude-status.sh
  • Stop Claude:
    docker stop claude-simple
💡 Pro Tip: Use multiple browser tabs to manage different Claude sessions. Bookmark each project's Claude terminal for quick access.

🔔 Mobile Notifications

To get notifications on your phone:

  1. Set up a webhook service (Pushover, Telegram Bot, IFTTT)
  2. Run the monitor with your webhook URL
  3. Get notified when Claude asks for permission or completes tasks
