# Archived Prototype Pages

This directory contains prototype and playground pages that were useful during development but are not part of the main application.

## ComponentPlayground-phase3.tsx

**Created**: Phase 3 of Git Diff Viewer Enhancement
**Purpose**: Interactive playground for testing the new UI components:
- StatusIcon (priority-based file status indicators)
- StatusBadge (original multi-badge approach)
- StatusBadge2 (simplified badge approach)
- StatusIcon2 (view-based approach)
- ThreeStateToggle (Working Tree | All Changes | Base Branch)
- SearchInput (with file path highlighting)

**To resurrect**: 
1. Uncomment the import in App.tsx
2. Uncomment the route in App.tsx
3. Visit http://localhost:5173/prototype/components

**Why archived**: Components are now ready for production integration. The playground served its purpose for iterating on designs and demonstrating functionality.
